## 贡献内容

1. 下载本项目 & 安装基于 VitePress 的预览环境

    ```sh
    git clone git@github.com:2ndLA/English.git
    cd English

    yarn install
    yarn dev
    ```

2. 编辑内容
3. 提交 Pull request
